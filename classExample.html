void Example::first ( )
inline

A simple stub function to show how links do work.

Links are generated automatically for webpages (like http://www.google.co.uk) and for structures, like ExampleStruct. For typedef-ed types use #ExampleStruct. For functions, automatic links are generated when the parenthesis () follow the name of the function, like example(). Alternatively, you can use example.

Returns
NULL is always returned.
